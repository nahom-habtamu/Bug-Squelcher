# Domain Entities — Backend
## Bug Squelcher — Tech Debt Bounty Board (Intent 1)

---

## Entity: Bug

The single domain entity for this intent.

### Fields

| Field | Type | Constraints | Notes |
|---|---|---|---|
| id | string (UUID v4) | PK, auto-generated, immutable | Generated by Prisma (`@default(uuid())`) |
| title | string | Required, min 1 char after trim | Trimmed before persistence |
| stepsToReproduce | string | Required, min 1 char after trim | Newline-delimited steps; trimmed before persistence |
| severity | Severity enum | Required | P0 / P1 / P2 / P3 |
| status | BugStatus enum | Required, defaults to Open | Open / InProgress / WorksOnMyMachine (Prisma) |
| createdAt | DateTime | Auto-set on create, immutable | Never updated after creation |
| updatedAt | DateTime | Auto-updated on every write | Managed by Prisma (`@updatedAt`) |

### Prisma Schema

```prisma
model Bug {
  id               String   @id @default(uuid())
  title            String
  stepsToReproduce String
  severity         Severity
  status           Status   @default(Open)
  createdAt        DateTime @default(now())
  updatedAt        DateTime @updatedAt
}

enum Severity {
  P0
  P1
  P2
  P3
}

enum Status {
  Open
  InProgress
  WorksOnMyMachine
}
```

### API Shape (what the REST API returns)

```typescript
interface Bug {
  id: string
  title: string
  stepsToReproduce: string
  severity: 'P0' | 'P1' | 'P2' | 'P3'
  status: 'Open' | 'In Progress' | 'Works on My Machine'
  createdAt: string   // ISO 8601
  updatedAt: string   // ISO 8601
}
```

Note: `status` values differ between Prisma storage and API output — the service layer maps between them.

## Enum Mappings

### Status: API ↔ Prisma

| API Display String | Prisma Storage Value |
|---|---|
| `'Open'` | `'Open'` |
| `'In Progress'` | `'InProgress'` |
| `'Works on My Machine'` | `'WorksOnMyMachine'` |

### Severity: API ↔ Prisma

Severity values are identical in both layers — no mapping required.

| API String | Prisma Value |
|---|---|
| `'P0'` | `'P0'` |
| `'P1'` | `'P1'` |
| `'P2'` | `'P2'` |
| `'P3'` | `'P3'` |
